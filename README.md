# webAPI

## INSTALL

```
pip install -r requirements.txt
```

## RUN
```
python app.py
```

```

http://localhost:5000/MEMBER_TOTAL_WIN_AMOUNT/?MEMBER_ID=1001&GAME_ID=7499
```
